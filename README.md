# 🎮 LOTSGAME

みんなで遊べるブラウザゲームを **99本** あつめたスマホ向けポータルサイト。

**▶ https://legendmanbou.github.io/LOTSGAME/**

## あそびかた

1. リンクを開く(スマホOK・インストール不要・誰でも参加OK)
2. 左上の ⚙️ 設定から **なまえ** を決める
3. 誰か1人が **部屋を作る** → 出てきた4桁の部屋IDをみんなに伝える
4. ほかの人は **部屋に入る** でそのIDを入力
5. 誰かがゲームをタップすると「〇〇さんが△△をえらんだよ!」が全員に届く → 同じゲームに集合!

部屋機能は PeerJS(WebRTC)による P2P 通信で、サーバー不要・GitHub Pages だけで動く。

## しくみ

- `index.html` / `css/style.css` — 画面(横3列 × 33行 = 99項目のグリッド)
- `js/games.js` — ゲーム99本のデータ
- `js/thumbs.js` — 公式サムネイル(OGP画像)のURL。自動生成ファイル
- `js/app.js` — 一覧表示 + 部屋機能(PeerJS)

## サムネイルの更新

ゲームを追加・変更したら:

```
node tools/fetch-thumbs.mjs
```

各公式サイトから OGP 画像URLを取り直して `js/thumbs.js` を書き換える。
取得できないサイトはファビコン → 絵文字カードの順で自動フォールバック。
